# Theme Background Images

This directory contains background images for each theme in the Lost & Found application.

## Required Image Files

Place the following image files in this directory:

1. **purple-bg.jpg** - Background image for Purple Gradient theme
2. **blue-bg.jpg** - Background image for Ocean Blue theme
3. **green-bg.jpg** - Background image for Forest Green theme
4. **orange-bg.jpg** - Background image for Sunset Orange theme

## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 1920x1080 or higher
- **Aspect Ratio**: 16:9 recommended
- **File Size**: Keep under 500KB for optimal performance

## Image Style Recommendations

Choose images that complement each theme's color palette:

- **Purple Gradient**: Abstract purple/violet patterns, night sky, galaxies
- **Ocean Blue**: Ocean waves, water, sky, beaches
- **Forest Green**: Nature, forests, leaves, landscapes
- **Sunset Orange**: Sunsets, warm tones, autumn scenes

## Example Sources for Free Images

- [Unsplash](https://unsplash.com) - Free high-quality stock photos
- [Pexels](https://pexels.com) - Free stock photos and videos
- [Pixabay](https://pixabay.com) - Free images and videos

## How It Works

Background images are displayed at 15% opacity behind the main content area, creating a subtle visual enhancement without interfering with readability.

The application automatically loads the appropriate background image based on the selected theme.
